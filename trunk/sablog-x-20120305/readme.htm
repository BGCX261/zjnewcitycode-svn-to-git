全新安装

上传所有文件到空间，设置以下目录和文件的权限：

  • ./data 目录以及子目录权限为 0777 (*nix系统)
  • ./attachments 目录权限为 0777 (*nix系统)
  • ./config.php 文件权限为 0777 (*nix系统)
访问/install目录。程序会自动引导并提示安装过程。

1.6 to 2.0 升级操作

上传全部文件。请注意以下两点：

  • 手工设置config.php文件。
  • 手工设置.htaccess文件。
  • 目录权限参考全新安装。

访问http://yourname/blog/install，选择1.6到2.0升级步骤，程序将自动进行而不用人工干预。升级完毕后到后台仔细重新设置一下配置。重建数据并更新所有缓存即完成整个升级过程。
